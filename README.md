# Project 05: Data Collection & Storage

Mục tiêu chính:

Store Data vào MongoDB thông qua việc tải file dữ liệu vào bucket

Khám phá dữ liệu

Chuyển đổi/Lưu địa chỉ IP từ hệ thống thành dữ liệu thành phố

Lọc dữ liệu qua hành vi người dùng để lấy product_id và url

Crawl tên sản phẩm với mỗi product_id và url tương ứng. Store vào db/Extract ra file

Set up:

Khởi tạo Project mới trên GCP console để quản lý tài nguyên.

Tạo Bucket project_5 để lưu trữ dữ liệu

Khởi tạo VM Instance (Ubuntu 22.04 LTS, US).

Cấu hình Firewall mở cổng 27017 (MongoDB) và 22 (SSH).

Set up MongoDB Community Server
